# 🔐 Secure Password Manager

A command-line password manager built using Python that allows users to securely store and manage their account credentials.

## 🚀 Features

- ➕ Add New Password
- 📋 View Saved Passwords
- 🔍 Search Password by Website
- 🗑️ Delete Password
- 💾 Store passwords in a text file
- 🖥️ Menu-driven interface

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
secure-password-manager/
│
├── password_manager.py
├── passwords.txt
└── README.md
```

## ▶️ How to Run

1. Open `password_manager.py` in Pydroid 3.
2. Run the program.
3. Choose an option from the menu.
4. Add, view, search, or delete passwords.
